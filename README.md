pra rodar na máquina, é necessário ter o nodejs instalado (de preferência, o mais recente)
pra ver funcionando, pode ser usado um emulador android como genymotion (o que eu uso), bluestacks, android studio

pra instalar, basta abrir um terminal/cmd na pasta do projeto, e rodar npm install
depois que terminar o *npm install*, rodar o *npm run android*

se estiverem com o emulador rodando, o aplicativo vai compilar e ser exibido no emulador. Todas as alterações feitas no código, são atualizadas em tempo real.

se preferir testar pelo celular, vai aparecer um QRCode no terminal, dai vcs vão baixar um app chamado  *Expo* na *playstire/appstore*, dai é só ler o QRCode, que vai dar pra ver o app rodando direto no celular.
